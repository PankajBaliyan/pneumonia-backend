-> how to run app:
uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload

-> build docker image:
docker build -t pneumonia-api .

-> run docker image:
docker run -p 8000:8000 pneumonia-api
Access http://localhost:8000/docs in your browser to test the /predict endpoint with an image upload.
